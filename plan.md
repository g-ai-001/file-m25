# 文件管理器 (file-m25) 项目规划

## 项目概述
基于Android的本地文件管理器应用，支持文件浏览、搜索、创建文件夹、重命名、删除等基础功能。

## 技术栈
- Kotlin 2.3.20 + Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt依赖注入
- Room数据库
- DataStore键值对存储
- Coil图片加载
- Navigation Compose导航
- 最小/最大SDK: API 36
