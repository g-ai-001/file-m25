# 文件管理器 (file-m25)

一个简洁高效的Android文件管理器应用。

## 功能特性

- 文件浏览：支持目录导航和文件列表展示
- 文件操作：创建文件夹、重命名、删除、复制、移动
- 排序功能：支持按名称、大小、日期排序
- 视图切换：列表视图和网格视图
- 文件搜索：快速搜索文件
- 文件详情：查看文件大小、日期、类型等信息
- 多选操作：支持批量选择和操作
- 存储空间：实时显示设备存储使用情况
- 简体中文：完整的中文界面支持
- 收藏功能：收藏文件/文件夹方便快速访问
- 主题自定义：自定义主题颜色和动态颜色
- OTA更新：支持本地APK文件检测和更新安装

## 技术栈

- Kotlin 2.3.20 + Jetpack Compose
- Material 3 设计语言
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- DataStore 偏好设置
- Navigation Compose
- 最小/目标SDK: API 36
