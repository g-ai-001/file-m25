# 文件管理器 (file-m25)

一个简洁高效的Android文件管理器应用。

## 功能特性

- 文件浏览：支持目录导航和文件列表展示
- 文件操作：创建文件夹、重命名、删除、复制、移动
- 排序功能：支持按名称、大小、日期排序
- 视图切换：列表视图和网格视图
- 文件搜索：快速搜索文件
- 文件详情：查看文件大小、日期、类型等信息
- 多选操作：支持批量选择和操作
- 存储空间：实时显示设备存储使用情况
- 简体中文：完整的中文界面支持

## 技术栈

- Kotlin 2.3.20 + Jetpack Compose
- Material 3 设计语言
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- Navigation Compose
- 最小/目标SDK: API 36
